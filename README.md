**Keybinds:**
- F to give pickaxe
- G to remove pickaxe
- Left click on geysirs to mine
<br>

**How to install:**
- go to releases
- pick the right version for your operating system
- unzip the .zip file
- execute the .exe file on windows or on linux the other thingy
- Done ✔️
